# Exam Hall Seating Optimizer - Proposed System

## Overview
This project allocates students to exam benches in classrooms using:
- Graph coloring for conflict minimization
- Predictive alerts for benches with potential conflicts
- Interactive 2D visualization per classroom

## Folder Structure
- `data/` → Input CSVs (`students.csv`, `classrooms.csv`)
- `output/` → Generated CSV seating plan
- `scripts/` → Main Python script `exam_seating_proposed.py`
- `requirements.txt` → Python dependencies
- `README.md` → Project description
